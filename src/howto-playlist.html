How-To Make a Playlist

This guide shows you how to make your own playlist for the 2AMetronome app.

To keep things simple (for me as the programmer) the app offers no functionality to create or manage a playlist. Instead it relies on a simple text file in the JSON format. JSON may be already familiar to you, but if not: don't worry. It is easy to learn.

Example Playlist

Here you see an example for a playlist file:

{
  "title": "Demo Playlist",
  "songs": [
    {
      "title": "Echoes of Tomorrow",
      "bpm": 128,
      "duration": "3:26"
    },
    {
      "title": "Dancing Shadows",
      "bpm": 165,
      "autoStop": 10,
      "duration": "2:35"
    },
    {
      "title": "Silent Symphony",
      "bpm": 72,
      "autoSilence": 10,
      "duration": "3:31"
    }
  ]
}
my-playlist.json

You could start a simple text editor, like Notepad on a Windows PC, copy the contents above into the notepad and save the file as my-playlist.json - et voil√†: you have made a valid playlist to load into the 2AMetronome app.

It is important to use a text application (like Notepad) and not a word processor (like Word), since a plain text file is required. You can also use online text editors like JSON Editor Online.

How the JSON format works is explained in detail here.

However, to make a playlist you don't have to read the manual. Let's have a look at the example above and see what's in the file. The playlist has a title, which is "Demo Playlist". It defines a list (or an array) with three songs. The list starts with a [ character and ends with a ] character. Every song in the list is followed by a comma , character - except the last one.

Song properties

Each song requires at least these two properties: title and bpm. Note that every property (the word on the left side of the colon :) in the file must start and end with a double quote ". The following properties are optional, here's what they do:

  • duration: Format: m:ss (minutes:seconds), the play time of the song. 2AMetronome sums the duration of each song and shows the overall playtime of your playlist.
  • autoStop: number of bars. 2AMetronome stops automatically after that number of bars.
  • autoSilence: number of bars. 2AMetronome makes no more noises after that number of bars but continues to blink, to give you visual feedback.

Load a Playlist on a Smartphone

In case you wonder how to load your playlist file on a smartphone: you could send the file by e-mail, or with a messenger app to your smartphone. On your smartphone, store the playlist file on the file system (for instance in the Download folder). Or you could store the file in Google Drive, Dropbox or another cloud file storage, and choose the file from that location.
